Resumes

Upload a new Resume

This feature allows you to upload a new resume into the system for analysis. (Supported formats: PDF, DOC, DOCX, TXT)

Download the Resume

This feature allows you to download the resume in the service company's format.

Add a Professional Experience

When a consultant completes an assignment after their resume has been imported, this application offers the ability to add this new experience. Once the changes are complete, the resume can be reprinted in Word format.

Update a Professional Experience

You may want to detail one or more experiences further after importing the resume. The application allows you to modify them and then reprint the updated resume in Word format.

Delete a Professional Experience

After importing a candidate's resume into the system, it is possible to delete one of their professional experiences.

Translate the Resume

Once the resume is imported into the system, it can be translated into five different languages. New languages can be integrated at the client's request.

Add/Update Certifications

After importing the resume, the system attempts to identify each certification. If it fails, it indicates 'to be completed' so that sales managers can select the correct title or add a new one. This helps track the number of certified individuals in the statistics.

Update Contact Information

Once the candidate's resume is imported into the system, it is possible to modify their personal contact information.

Update Degrees

After importing a candidate's resume, it is possible to manage their list of degrees.

Update Spoken Languages

Once the candidate's resume is imported, it is possible to modify the list of languages they speak.

Update Technical Skills

After importing the resume, it is possible to modify the technical skills section located in the document's header.

Upload and Update Documents

This feature allows for bulk adding of personal documents related to the candidate. These files can be automatically attached when sending their application.

Send an Interview Invitation

Sales managers can schedule an interview in a few clicks, specifying the job posting that prompted the invitation.

Job Postings

Upload a Job Offer

Allows uploading a client's job description (formats: PDF, DOC, DOCX, TXT). This action triggers a compatibility score calculation with existing resumes. The best profiles are reported by e-mail to the sales team.

Update and Download the Offer Description

After import, the description is pre-formatted into editable sections to adapt the content before exporting to Word format.

Optimize a Resume for a Job Offer

A compatibility score is calculated between a job posting and resumes. The most relevant profiles are displayed, and it's possible to optimize a resume for this offer, which will create a new, editable version exportable to Word.

Send the Application File to the Client

Once a resume is optimized for an offer, a 'Send Application to Client' button appears. It allows creating a pre-formatted email with the optimized resume and the candidate's basic documents as attachments.

Generate a Quiz for a Job Offer

Once the job offer is analyzed, the application provides the option to generate a technical quiz by pre-selecting the main topics of the offer. It is possible to change them and define the number of questions per topic.

Update Quiz Questions

Once the quiz for an offer has been generated, it is possible to moderate the questions. You can add, delete, or update existing questions. It will also be possible to modify the time available for candidates to answer it.

Have a Candidate Take the Quiz

Once the quiz is generated, candidates applying for the job offer can take the test online. They will receive a secure link in their email inbox to take the test, and the results will be sent to the user who sent the invitation.

Quiz Score Ranking

Once the candidate has finished the quiz, their score will be saved in the database and available for viewing. A list of scores from candidates who have taken the quiz will be available.

Invoices and Support

View and Download Invoices

This option allows you to view the status of your invoices and download them.

Contact Support

Use this option to ask a question or report a bug to our support team.

Cancel Subscription

This option allows you to cancel your subscription.

Statistics and Settings

Account Settings

This option allows you to change your password or update your company information, some of which appears in emails.

Statistics

Displays statistics for your candidates. Some data can be downloaded as an Excel report.
