Currículums

Subir un nuevo currículum

Esta función le permite cargar un nuevo currículum en el sistema para su análisis. (Formatos compatibles: PDF, DOC, DOCX, TXT)

Descargar el currículum

Esta función le permite descargar el currículum en el formato de la empresa de servicios.

Añadir una experiencia profesional

Cuando un consultor completa una asignación después de que su currículum ha sido importado, esta aplicación ofrece la posibilidad de añadir esta nueva experiencia. Una vez que los cambios están completos, el currículum se puede volver a imprimir en formato Word.

Actualizar una experiencia profesional

Es posible que desee detallar una o más experiencias después de importar el currículum. La aplicación le permite modificarlas y luego volver a imprimir el currículum actualizado en formato Word.

Eliminar una experiencia profesional

Después de importar el currículum de un candidato en el sistema, es posible eliminar una de sus experiencias profesionales.

Traducir el currículum

Una vez que el currículum se importa en el sistema, se puede traducir a cinco idiomas diferentes. Se pueden integrar nuevos idiomas a petición del cliente.

Añadir/Actualizar certificaciones

Después de importar el currículum, el sistema intenta identificar cada certificación. Si falla, indica 'a completar' para que los gerentes de ventas puedan seleccionar el título correcto o añadir uno nuevo. Esto ayuda a hacer un seguimiento del número de personas certificadas en las estadísticas.

Actualizar la información de contacto

Una vez que el currículum del candidato se importa en el sistema, es posible modificar su información de contacto personal.

Actualizar títulos

Después de importar el currículum de un candidato, es posible gestionar su lista de títulos.

Actualizar idiomas hablados

Una vez que se importa el currículum del candidato, es posible modificar la lista de idiomas que habla.

Actualizar habilidades técnicas

Después de importar el currículum, es posible modificar la sección de habilidades técnicas ubicada en el encabezado del documento.

Subir y actualizar documentos

Esta función permite añadir de forma masiva documentos personales relacionados con el candidato. Estos archivos se pueden adjuntar automáticamente al enviar su candidatura.

Enviar una invitación a una entrevista

Los gerentes de ventas pueden programar una entrevista en unos pocos clics, especificando la oferta de trabajo que motivó la invitación.

Ofertas de empleo

Subir una oferta de empleo

Permite subir la descripción de un puesto de trabajo de un cliente (formatos: PDF, DOC, DOCX, TXT). Esta acción desencadena un cálculo de la puntuación de compatibilidad con los currículums existentes. Los mejores perfiles se comunican por correo electrónico al equipo de ventas.

Actualizar y descargar la descripción de la oferta

Después de la importación, la descripción se preformatea en secciones editables para adaptar el contenido antes de exportarlo a formato Word.

Optimizar un currículum para una oferta de empleo

Se calcula una puntuación de compatibilidad entre una oferta de empleo y los currículums. Se muestran los perfiles más relevantes y es posible optimizar un currículum para esta oferta, lo que creará una nueva versión editable y exportable a Word.

Enviar el archivo de candidatura al cliente

Una vez que un currículum está optimizado para una oferta, aparece un botón 'Enviar candidatura al cliente'. Permite crear un correo electrónico preformateado con el currículum optimizado y los documentos básicos del candidato como archivos adjuntos.

Generar un cuestionario para una oferta de empleo

Una vez analizada la oferta de empleo, la aplicación ofrece la opción de generar un cuestionario técnico preseleccionando los temas principales de la oferta. Es posible cambiarlos y definir el número de preguntas por tema.

Actualizar las preguntas del cuestionario

Una vez que se ha generado el cuestionario para una oferta, es posible moderar las preguntas. Puede añadir, eliminar o actualizar las preguntas existentes. También será posible modificar el tiempo disponible para que los candidatos lo respondan.

Hacer que un candidato realice el cuestionario

Una vez generado el cuestionario, los candidatos que se postulen a la oferta de empleo pueden realizar la prueba en línea. Recibirán un enlace seguro en su bandeja de entrada de correo electrónico para realizar la prueba, y los resultados se enviarán al usuario que envió la invitación.

Clasificación de puntuaciones del cuestionario

Una vez que el candidato ha terminado el cuestionario, su puntuación se guardará en la base de datos y estará disponible para su consulta. Estará disponible una lista de las puntuaciones de los candidatos que han realizado el cuestionario.
